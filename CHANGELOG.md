# Changelog

All notable changes to QR Gen.

## [1.1.0] - 2026-02-25

### Added
- **PWA Support** - Install as a standalone app on mobile/desktop
- **Custom Favicon** - SVG favicon with QR code icon
- **Social Meta Tags** - Better sharing previews
- **Manifest** - Full app manifest for installability

## [1.0.0] - 2026-01-27

### Added
- **Initial Release**
- Multiple data type support:
  - Plain text
  - URLs
  - Email addresses
  - Phone numbers
  - WiFi credentials
- Size customization options
- Error correction level selection
- Real-time QR code preview
- PNG download functionality
- Clean responsive UI
- Works offline (no external API calls)
