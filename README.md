# QR Gen

Fast QR code generator for text, URLs, and data.

## Features

- **Multiple Data Types**
  - Plain text
  - URLs
  - Email addresses
  - Phone numbers
  - WiFi credentials
- **Customization**
  - Size options
  - Error correction levels
- **Download** - Save as PNG image
- **Preview** - Real-time QR generation

## Usage

1. Open `index.html` in a browser
2. Enter text or URL
3. QR code generates instantly
4. Download or screenshot

## Common Uses

- Share URLs without typing
- WiFi password sharing
- Business card contact info
- Event check-in codes

## Tech

- Pure HTML/CSS/JavaScript
- No external API calls
- Works offline
